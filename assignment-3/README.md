# Assignment 3 - Dynamic Product Catalog Integration

Khaadi e-commerce site with MongoDB integration, server-side pagination, search, category filtering, and price range filtering.

## Tech Stack
- Node.js + Express
- EJS Templates
- MongoDB + Mongoose

## How to run
1. npm install
2. node seed.js (seed the database)
3. node server.js
4. Visit http://localhost:3000/products
